## 5. 核心设计分析

### 5.1 调用链

```
REPL.tsx:onQueryEvent(event)
    └→ handleMessageFromStream(event, callbacks...)
        ├→ onMessage(message)           // 完整消息回写
        ├→ onSetStreamMode(mode)        // spinner 模式切换
        ├→ onStreamingToolUses(update)  // 流式工具调用
        ├→ onStreamingThinking(update)  // 流式思考
        ├→ onStreamingText(update)      // 流式文本
        └→ onApiMetrics(metrics)        // TTFT 等指标
```

### 5.2 设计模式

- **回调模式**（观察者变体）：事件处理与 UI 更新解耦
- **函数式更新**：`f: (current) => next` 形式保证并发安全

### 5.3 事件处理分支

| 事件类型 | 处理动作 |
|---------|---------|
| `stream_request_start` | 设置 `requesting` 模式 |
| `content_block_start` | 按 block 类型设置模式 |
| `content_block_delta` | 累加文本/JSON |
| `message_stop` | 清空工具调用列表 |
| `message_delta` | 设置 `responding` 模式 |
