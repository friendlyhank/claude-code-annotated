## 11. 实现原理与验证方式

- **实现原理**：
  1. 使用 `anthropic.beta.messages.create({ stream: true })` 获取原始流
  2. 遍历流事件，按事件类型更新 contentBlocks 和 usage 状态
  3. 每个 content_block_stop 时 yield AssistantMessage，允许 UI 逐步渲染
  4. 所有事件都同时 yield StreamEvent，保留原始事件供上层消费

- **验证方式**：
  ```bash
  bun run typecheck  # ✅ 通过
  bun run build      # ✅ 通过
  ```
