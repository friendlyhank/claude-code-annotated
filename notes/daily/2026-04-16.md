## 知识点

1. **glob 模式匹配**：
   - `**` 匹配任意路径（包括路径分隔符）
   - `*` 匹配非路径分隔符
   - `?` 匹配单个非路径分隔符

2. **工具执行链路**：
   - findToolByName → inputSchema.safeParse → validateInput → canUseTool → tool.call → mapToolResult

3. **相对路径匹配**：
   - ripgrep 使用相对路径匹配 glob 模式
   - 需要计算文件相对于搜索目录的相对路径

4. **工具注册流程**：
   - getAllBaseTools() 是所有内置工具的唯一真相源
   - hasEmbeddedSearchTools() 决定是否包含 Glob/Grep

5. **API 工具传递**：
   - tools 必须转换为 API 格式（name, description, input_schema）
   - ToolUseContext.options.tools 必须正确填充
   - getTools(permissionContext) 从权限上下文获取可用工具
